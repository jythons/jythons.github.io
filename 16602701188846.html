不保留本地修改

2022/08/12 posted in  Git

1.背景

当前分支代码做了一些没用的修改甚至还commit了，不保留本地的修改。

2.操作

  • 先同步远端信息
git fetch --all
  • 可以通过将本地的git指针指向远端某个分支
git reset --hard origin/远端分支名
  • 再pull更新本地代码
git pull origin 远端分支名
