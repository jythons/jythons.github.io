负载均衡之url_hash

2020/10/28 posted in  nginx

从字面上可以看出，他是基于URL的hash算法，根据URL获得的hash值将请求分配到不同的节点上

##Nginx配置

upstream tomcats {
        hash $request_uri; 
        server 192.168.3.26:8080;
        server 192.168.3.27:8080;
        server 192.168.3.28:8080;
}

server {
        listen 80;
        server_name www.tomcats.com;

        location / {
                proxy_pass http://tomcats;
        }
}
